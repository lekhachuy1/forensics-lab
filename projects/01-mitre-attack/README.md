# Project 01: MITRE ATT&CK for Forensics & SOC

## Goal

Understand MITRE ATT&CK and apply it to digital forensics, SOC investigation, CTF forensics, and incident response.

## What is MITRE ATT&CK?

MITRE ATT&CK is a globally accessible knowledge base of adversary tactics and techniques based on real-world observations.

## Learning Objectives

- Understand tactics, techniques, and sub-techniques
- Learn how to read the Enterprise ATT&CK Matrix
- Map Windows Event Logs to ATT&CK techniques
- Map PCAP/network behavior to ATT&CK techniques
- Build a small ATT&CK-based investigation report
